# 理论题作答助手

一个本地运行的 Web UI 小工具：

- 把单选/多选题统一贴到一个输入框
- 同时调用多个已登录的平台网页端提问
- 自动提取各平台答案并做重合度统计
- 支持平台勾选、任务队列、强制结束、原始回答查看

当前默认界面已经切到 Web UI，原来的 Tk 桌面界面仍保留为备用入口。

## 当前支持的平台

- DeepSeek
- 千问国内版
- 千问
- 智谱清言
- Kimi
- 腾讯元宝

## 运行方式

建议先创建虚拟环境：

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m playwright install chromium
```

启动 Web UI：

```powershell
python webui.py
```

或者直接双击：

- `启动助手.bat`

启动后会自动打开本机地址，默认类似：

- `http://127.0.0.1:8765`

如果你想临时使用旧版 Tk 桌面界面：

```powershell
python app.py
```

## 使用流程

1. 勾选本次要使用的平台
2. 点 `打开网页会话`
3. 在弹出的浏览器里完成各平台登录
4. 回到 Web UI，把题目加入队列

多道题可以一次性导入，题目之间用单独一行 `---` 分隔，例如：

```text
1. 下列说法正确的是：
A. ...
B. ...
C. ...
D. ...
---
2. 逻辑学的创始人是：
A. ...
B. ...
C. ...
D. ...
```

## 界面说明

- 左侧：输入题目、勾选平台、加入队列
- 中间/下方：任务队列，显示待执行、运行中、已完成、已失败、已终止
- 右侧：当前选中任务的各平台状态、推荐答案、原始回答和运行日志

`强制结束` 会：

- 终止当前正在运行的任务
- 把还没开始的待执行任务从队列里移除

## 关键文件

- `webui.py`：本地 Web 服务入口
- `webui_static/`：Web UI 页面、样式、脚本
- `queue_service.py`：任务队列、状态同步、强制结束逻辑
- `automation.py`：浏览器自动化与答案抓取
- `providers.py`：平台地址与 selector 配置
- `consensus.py`：答案提取与投票统计
- `app.py`：旧版 Tk 桌面界面

## 已知限制

- 所有平台都需要你先在网页端登录
- 某些站点会改版，必要时要调整 `providers.py` 或 `automation.py`
- 网页端偶尔会出现风控、弹窗或页面结构变化，这类问题通常需要按平台单独兼容
